# Observability Plugin

Monitoring, tracing, and observability for AI agent interactions.

## Supported Backends

| Backend | Type | Features |
|---------|------|----------|
| Langfuse | Cloud/Self-hosted | Traces, prompts, eval |
| OpenTelemetry | Standard | Traces, metrics, logs |
| Phoenix (Arize) | Local | LLM observability |
| Weights & Biases | Cloud | Experiment tracking |

## Features
- Conversation tracing and replay
- Token usage and cost tracking
- Latency monitoring
- Quality evaluation scoring
- Prompt version management
- A/B testing support
