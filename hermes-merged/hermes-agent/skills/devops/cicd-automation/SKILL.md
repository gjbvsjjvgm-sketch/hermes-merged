# CI/CD Automation

DevOps automation for building, testing, and deploying applications.

## Capabilities

- **Docker**: Build, manage, and orchestrate containers
- **GitHub Actions**: Create and manage CI/CD workflows
- **Infrastructure as Code**: Terraform, Pulumi, CloudFormation
- **Deployment**: Deploy to AWS, GCP, Azure, Vercel, Netlify
- **Monitoring**: Set up logging, alerting, and observability
- **Security Scanning**: SAST, DAST, dependency scanning

## Usage Examples

User: "Create a Dockerfile for my Node.js app"
→ Generate optimized Dockerfile with multi-stage build

User: "Set up a GitHub Actions workflow for CI"
→ Create workflow YAML with test, lint, build steps

User: "Deploy my app to AWS"
→ Generate Terraform config and deployment scripts

User: "Set up monitoring for my service"
→ Configure health checks, logging, and alerting
