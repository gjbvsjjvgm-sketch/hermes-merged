# Data Analysis

Comprehensive data analysis toolkit - load, clean, analyze, and visualize data with AI assistance.

## Capabilities

- **Data Loading**: CSV, Excel, JSON, SQL databases, APIs
- **Data Cleaning**: Handle missing values, outliers, duplicates
- **Statistical Analysis**: Descriptive stats, correlations, regression
- **Visualization**: Charts, plots, dashboards (matplotlib, plotly)
- **Machine Learning**: Classification, clustering, prediction
- **Report Generation**: Automated analysis reports with insights

## Usage Examples

User: "Analyze this CSV file and show me the trends"
→ Load data, compute statistics, create trend charts

User: "Create a dashboard from this data"
→ Generate interactive visualizations

User: "Find anomalies in this dataset"
→ Run anomaly detection algorithms

User: "Predict next quarter's sales"
→ Build and evaluate a forecasting model

## Supported Libraries
- pandas, numpy - Data manipulation
- matplotlib, seaborn, plotly - Visualization
- scikit-learn - Machine Learning
- scipy - Statistical analysis
- statsmodels - Econometric models
